Tombstone Tapes

Project objective: Create a website for a bespoke casette tape production company stylised after Windows 95.

Tombstone Tapes

Motivation

Inspiration

I was tasked with creating a bespoke website to accompany small business Tombstone Tapes. To fit with the retro feel of redundant data formats, we decided to look back to the past and create a website inspired by Windows 95.

Project Overview

Design

The design of the Tombstone Tapes homepage is inspired by Windows 95, with colours and fonts taken from the operating system.

This gave us the opportunity to include fun elements on the website to interact with, such as a fully functional start menu complete with icons to external sites, displayed in 16-bit Windows colour palette, along with a real time clock along the task bar.

The containers on the website are stylised after Windows 95 dialogue boxes, including image containers stylised after MS Paint containers.

The website is fully responsive to work on mobile devices, and likewise the start menu is still fully functional.

Conclusion

Personal Growth and Lessons Learnt

Tombstone Tapes was an exercise in creating a website with a specific design brief, and was a great opportunity to learn more about working with CSS and responsive design. Creating a bespoke interactive start menu allowed me to create some fun elements and experiment with some simple JavaScript.
